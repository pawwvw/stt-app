# ❓ Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Q: Почему при первом запуске нужно скачивать модель?
**A:** Чтобы уменьшить размер дистрибутива. Вместо ~150 MB вы скачиваете ~10-50 MB приложение + ~75 MB модель отдельно. Это также позволяет обновлять модель независимо от приложения.

### Q: Где хранится модель?
**A:** 
- **Windows**: `%APPDATA%\com.pawwvw.stt-app\models\ggml-tiny.bin`
- **Linux**: `~/.local/share/com.pawwvw.stt-app/models/ggml-tiny.bin`
- **macOS**: `~/Library/Application Support/com.pawwvw.stt-app/models/ggml-tiny.bin`

### Q: Можно ли использовать приложение без интернета?
**A:** Да! После того как модель скачана один раз, приложение работает полностью офлайн. Все обрабатывается локально на вашем компьютере.

### Q: Отправляются ли мои данные куда-то?
**A:** Нет. Все обрабатывается локально. Ваши аудио файлы и расшифровки никуда не отправляются.

## Установка

### Q: На Linux пишет "Whisper CLI не найден"
**A:** Вам нужно установить whisper.cpp:
```bash
git clone https://github.com/ggerganov/whisper.cpp
cd whisper.cpp
make
sudo cp main /usr/local/bin/main
```
Подробнее: [LINUX_MAC_INSTALL.md](./LINUX_MAC_INSTALL.md)

### Q: На macOS не запускается приложение
**A:** 
1. Установите whisper.cpp: `brew install whisper-cpp`
2. При первом запуске может потребоваться разрешить в System Preferences → Security & Privacy

### Q: Windows Defender блокирует установку
**A:** Это ложное срабатывание. Приложение безопасно. Вы можете:
- Нажать "More info" → "Run anyway"
- Добавить в исключения Windows Defender
- Собрать из исходников самостоятельно

## Использование

### Q: Какие форматы аудио поддерживаются?
**A:** MP3, WAV, OGG, M4A, WebM, FLAC

### Q: Сколько времени занимает расшифровка?
**A:** Зависит от длительности аудио и мощности процессора. Примерно:
- 1 минута аудио = 5-30 секунд обработки
- Первая расшифровка может быть медленнее (загрузка модели в память)

### Q: Можно ли расшифровывать на других языках?
**A:** В текущей версии настроен только русский язык. В будущих версиях будет добавлена поддержка других языков.

### Q: Почему расшифровка неточная?
**A:** Возможные причины:
- Плохое качество записи (шумы, музыка на фоне)
- Тихий голос
- Используется модель `tiny` (маленькая, но быстрая)

В будущем будет добавлена возможность выбора более точных моделей (base, small, medium).

## Технические вопросы

### Q: Сколько места занимает приложение?
**A:**
- Дистрибутив: ~10-50 MB
- Модель: ~75 MB
- Итого: ~85-125 MB

### Q: Какие системные требования?
**A:**
- **Процессор**: любой современный CPU (2010+)
- **Память**: минимум 2 GB RAM
- **Место**: ~200 MB свободного места
- **ОС**: Windows 10+, Ubuntu 20.04+, macOS 10.15+

### Q: Как удалить модель?
**A:** Просто удалите файл `ggml-tiny.bin` из директории данных приложения (см. выше). При следующем запуске приложение предложит скачать её снова.

### Q: Как обновить модель?
**A:** Удалите старую модель и нажмите "Установить модель" снова в приложении.

## Проблемы и ошибки

### Q: Ошибка "Не удалось запустить Whisper CLI"
**A:** 
- **Windows**: Убедитесь что все DLL файлы на месте
- **Linux/Mac**: Установите whisper.cpp (см. инструкции выше)
- Попробуйте перезапустить приложение

### Q: Ошибка "Модель не найдена"
**A:** Нажмите кнопку "Установить модель" в интерфейсе приложения.

### Q: Приложение зависает при расшифровке
**A:**
- Подождите дольше - большие файлы обрабатываются медленно
- Попробуйте более короткий аудио файл
- Перезапустите приложение

### Q: Не удается скачать модель
**A:**
- Проверьте интернет-соединение
- Отключите VPN/прокси
- Проверьте место на диске (~200 MB свободно)
- Попробуйте позже (возможны проблемы с HuggingFace)
